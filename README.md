# Home Assistant Add-on Repository: Rathole

Home Assistant add-on for [Rathole](https://github.com/rapiz1/rathole) — a lightweight reverse proxy for NAT traversal.

## Add-ons

### [Rathole Client](./rathole-client)

Runs the rathole client to tunnel your Home Assistant instance through a remote rathole server for secure remote access without port forwarding.

## Installation

1. Open your Home Assistant instance
2. Navigate to **Settings** → **Add-ons** → **Add-on Store**
3. Click the menu (⋮) in the top right → **Repositories**
4. Add this repository URL
5. Find and install **Rathole Client**
